# 高雄兩天一夜 並行 Agent 調度提示詞（YouTube 取材版）

## 旅遊基本資料
- 日期：2026/7/4(六) ~ 7/5(日)，兩天一夜（**盛夏**：炎熱、午後雷陣雨、可能颱風）
- 旅客：**1 大人 + 1 兩歲幼兒（需推車）**，單獨帶娃、無人接手
- 出發：台中 → 高鐵左營（約 50 分）→ 鹽埕基地，原路往返
- 住宿：**鮪魚家族飯店鹽埕館**（鹽埕區莒光街200號，近橘線 O2 鹽埕埔站／輕軌 C9 鹽埕埔站），1 晚不換房
- 市區交通：**全程大眾運輸**（高雄捷運橘/紅線、環狀輕軌、公車），**不搭計程車、不自駕**
- 行程重點：在地美食/夜市、地標打卡/戶外、輕鬆慢步調

## 🌟 核心規則（所有 agent 一律遵守）

1. **YouTube 取材 — 來源分級 A/B/C**：每一個排入正式行程的景點/餐廳，必須標來源等級：
   - **A**：影片標題或可穩定取回的頻道資訊**直接**佐證該店名/景點。
   - **B**：YouTube 影片主題相符，但逐點細節（店名、地址）由圖文部落格（peipei.tw / bobowin.blog / kenalice.tw 等）補齊 → **須同列 YouTube 連結與部落格網址**，並註「細節未經影片逐幀核實」。
   - **C**：僅部落格、無對應 YouTube 影片 → **只能進「待查證候選」區，不得排入正式行程**。
2. **連結真實性閘門（防幻覺）**：每條 YouTube 連結寫入前，**必須以 WebFetch 實際開啟**，確認取回標題＋頻道且主題相符；開不了或對不上者，一律降為文字描述（頻道名＋搜尋關鍵字）並標「連結未驗證」。**明文禁止**：不得自行組裝/猜測 video id、不得把部落格內容掛上 YouTube 連結冒充來源、標題未含店名者不得宣稱「YouTube 已確認店名」。`seed-youtube.md` §B 已給的 video id（如 9jct6n7vVZ8）也須逐一開啟確認仍存在。
3. **Maps 僅佐證**：Google Maps MCP 只用來查證「最近捷運/輕軌站、無障礙電梯、步行距離/時間、營業時間、座標」，**不可作為景點來源**。使用前先讀根目錄 [`../../../CLAUDE.md`](../../../CLAUDE.md) 的「已知問題與正確用法」——`maps_place_details` 須用 **`placeId`（駝峰式）**、兩步驟查詢（先 `maps_search_places` 取 id）。
4. **單獨帶兩歲娃作息**：動線要短、可隨時放棄；**午睡時段（約 13:00-15:30）**安排回飯店或室內冷氣點；避免長時間排隊與長距離連走；每段步行盡量 ≤10 分。
5. **盛夏雨備**：每個戶外點都要配一個**鄰近室內冷氣備案**；雨備須涵蓋**撤退途中**（以推車雨罩＋雨傘護住幼兒），不只終點室內。標出室內避暑點與推車友善度。
6. **推車無障礙**：路線以「有電梯捷運站＋低地板輕軌＋平路」為主；遇爬坡/階梯點（領事館登山、龍虎塔、旗津登高）明確標註並給替代。
7. **主選＋備案，且高風險點須「整段替代」**：景點與餐廳都給「主選 + 1 備案」。凡含「可能整個不開放」風險者——**渡輪/愛之船僅週末假日、哈瑪星小火車 2 歲可否未定、需排隊的熱門點**——必須在該點旁直接附**「若不開放/排隊過長的整段替代方案」**（換成另一段行程），而非只給同類備案。
8. **時效查證**：凡寫入手冊者標注「出發前(2026/7/4-5)須再查證」（夜市公休、渡輪/愛之船班次、小火車票價與2歲可否、輕軌票價、特展檔期、高鐵班次）。
9. **起手種子**：先讀 [`seed-youtube.md`](seed-youtube.md)（已偵查彙整的查詢/頻道/種子景點/實務 tips/須剔除誤導項），再沿頻道延伸驗證。

## 各 Agent 詳細任務

### Agent 1 — 路線統籌（`route_youtube.md`）
找 YouTuber 實拍的「高雄兩天一夜／不開車／親子」影片，歸納動線，**重排成適合單獨帶兩歲娃＋盛夏的作息式行程**。
- 產出 **2 套兩天動線**（主選＋備案），時間軸：上午（趁涼戶外）→ 午餐（鹽埕基地附近）→ 午睡（13:00-15:30）→ 傍晚（退暑後戶外/地標）→ 晚餐/夜市；第二天上午主景點 → 午餐 → 返程緩衝 → 左營高鐵。
- **必填①：抵達日午睡落地方案**（check-in 15:00 vs 午睡 13:00-15:30 的矛盾，三選一並標可行性）：(a) early check-in 可能（轉 Agent 6 向飯店查證可否付費/視空房提早入住、行李能否先寄櫃台）；(b) 室內冷氣點推車內補眠（明指棧貳庫/鐵道館/咖啡座哪些容許推車停著讓幼兒睡）；(c) 接受首日午睡縮短、改於交通工具上補眠並把下午改靜態。
- **必填②：第二天午睡缺口**：主行程壓 09:00-12:00 趁涼且不拖行李，13:00 後回飯店取行李 → 直奔左營，**午睡改高鐵車上補**（行程表標「幼兒預計在高鐵上午睡」）；退房後不再安排需久留的室內點。
- **必填③：撤退/最短收工**：每套動線附「最短收工路徑」（任一時段從當前點 1-2 步回 C9/O2 鹽埕埔 → 飯店的方式與分鐘數）；每個點標「必看 / 娃累可略過」。
- 每個時段點標：YouTube 出處＋來源等級、最近站、步行分鐘、室內/戶外、雨備。
- 明確排除純大眾運輸＋帶推車 CP 值低的市郊大點（義大/夢時代/草衙）並說明原因。

### Agent 2 — 鹽埕基地景點/地標（`spots_yancheng.md`）
聚焦「飯店步行或一站輕軌可達」：駁二（挑1-2倉庫）、哈瑪星台灣鐵道館＋小火車、棧貳庫 KW2、大港橋、西子灣、愛河（愛之船）。
- 每點：YouTube 出處連結＋頻道＋來源等級、為何適合兩歲娃、最近捷運/輕軌站與從飯店到達方式（Maps 佐證步行分鐘/電梯）、室內或戶外、盛夏注意、營業時間與票價（標待查證）。
- **整段替代**：愛之船/旗津渡輪僅週末假日（7/4-5 應有航班但須查證），小火車 2 歲可否未定——各須附「不開放時改去哪」。
- **旗津**特別標：無遮蔭＋避雨點少，**午後雷雨時段不排入**；只逛老街＋海灘、捨棄登高。
- ⚠️ 剔除：**棧貳庫旁白色旋轉木馬（2023/11 已停業）**、「愛河之心」（距鹽埕4公里，勿與鹽埕河岸混淆）——名稱欄不得出現旋轉木馬當賣點。
- 給「主選 3 點 + 備案 2 點」並排出鹽埕放射狀短程順序。

### Agent 3 — 在地美食/夜市（`food_nightmarket.md`）
鎖定帶娃可坐下、有座位、不必為吃飯特地曝曬移動的選項。
- **鹽埕步行圈**（最省力）：鹽埕老街單店小吃（脆皮煎餃、煙燻牛肋排、豬油拌麵、切仔麵、排骨飯）、大溝頂/新樂街、鹽埕第一公有市場、王家豆花。
- **夜市**：六合（美麗島直達、推車最友善、鎖定莊記海產粥等有桌椅老攤）主選；光華（有冷氣老店）雨天備案；瑞豐標擁擠＋需轉乘＋週一三休、列不推。
- **晚餐早收備案**：夜市時段（17:00-19:30）與兩歲就寢（目標 20:00 前）衝突 → 備案「鹽埕步行圈就近晚餐／外帶回房」，標往返步行 5-10 分。
- 每店：YouTube 出處＋來源等級、是否有座位/冷氣、帶推車可行性、建議時段、營業時間/公休（標待查證）。早午晚三餐都要有鹽埕步行圈方案。

### Agent 4 — 室內避暑/親子備案（`indoor_kids.md`）
建立「最熱時段（12:00-16:00）與午後雷雨」室內冷氣備案庫，**分兩層**：
- **第一層＝即時雨備（步行或一站輕軌可達，可隨時撤入）**：哈瑪星鐵道館、棧貳庫商場、駁二倉庫店家、木育森林、美麗島站光之穹頂、鹽埕第一公有市場（半遮蔭）。
- **第二層＝半日主目的地（非臨時撤退點）**：科工館——離捷運遠、需轉公車、班距疏，只當「半天主目的地」並標往返交通時間；**下雨不導向需久候公車的遠點**。
- 每點：YouTube 出處＋來源等級、適齡（2歲能玩什麼）、推車與哺乳/尿布設施、最近站與從鹽埕到達方式、票價（待查證；木育森林約400元/須陪同/推車寄放）。
- 廁所動線：標出「單人帶娃可整台推車進入的無障礙廁所」（美麗島出口4、駁二/棧貳庫商場）。

### Agent 5 — 交通實務（`transport.md`）
- **高鐵段（台灣高鐵官網/客服為準，非 YouTube）**：查證 **2 歲幼兒票規**（未滿2歲免票不佔位、滿2歲是否需購孩童票、佔位取捨）；列 7/4(六)上午台中→左營、7/5(日)下午左營→台中**實際可選班次**與是否建議事先訂位；嬰兒推車上高鐵規定（可否不收折、放置位置、行李規範）。
- **左營→鹽埕轉乘實測**：逐站列電梯/直達電梯位置、轉乘步行距離與是否需出閘，評估「單人帶推車＋2 件行李」在**美麗島轉乘**的實際可操作性（全程最吃力一段）；若吃力則比較更少轉乘的替代並給結論。
- **市區三段帶推車實務**：①鹽埕↔駁二/哈瑪星（輕軌 C9 一站）②鹽埕↔六合（橘線 O2→美麗島，地下免轉乘）③鹽埕↔西子灣（橘線 O2→O1 一站）。每段：站名、電梯位置、時間、是否轉乘。
- **票卡與刷卡**：一卡通、2歲免費（僅大人持卡進/出站各刷一次、幼兒免刷）；輕軌**無閘門**須自行在月台驗票機刷進/刷出（帶推車先停妥再刷），漏刷出站後果；班距、移動式避暑要點。
- **返程**：鹽埕→左營高鐵→台中，預留盛夏午後雷雨緩衝與行李寄放方案。

### Agent 6 — 飯店實務/天氣/醫療/費用（`summary.md`）
- **飯店實務查證（飯店官網/訂房平台/電話，非 YouTube）**：鮪魚家族飯店鹽埕館標準 check-in/out 時間、**check-in 前及退房後可否寄放行李**（首日 10:30 與返程動線的前提，務必確認）、有無嬰兒床/床欄/澡盆/嬰兒備品可借、房型與電梯是否方便推車進房。**若無法確認**，列入查證清單並標「動線前提，未證實前以左營站置物櫃為備案（標尺寸/費用）」。
- **颱風/豪雨 B 計畫**：①出發前查證來源（中央氣象署即時雷達、台灣高鐵/台鐵停駛公告、高雄市府停班課）②高鐵/台鐵「天災停駛之退改票規則」與當日查詢管道③一份「**完全不離開鹽埕步行圈**的純室內一日方案」（飯店＋鹽埕第一公有市場＋鄰近冷氣小吃＋木育森林/棧貳庫一站輕軌）④判斷是否取消/改期當日行程的決策點時間。
- **醫療/安全**：至少 1 家**有兒童夜間急診的具體院名**（如高醫/長庚，Maps 佐證最近站）＋24 小時藥局；走失預防（當日穿著照、寫大人手機識別卡、向站務/夜市服務台求助）。
- **哺乳/換尿布**：點地圖（飯店、美麗島出口4、駁二/棧貳庫場館；衛福部哺集乳室地圖）。
- **行前裝備**：推車＋背巾、**推車雨罩＋幼兒輕便雨衣**、防曬補水、幼兒備品；查高雄是否有推車租借作自帶備援。
- **費用估算（2人）**：高鐵往返、捷運輕軌、餐食、門票、住宿、雜支；**逐項標明 2 歲幼兒是否計費**（須與 Agent 5 票規一致）。

## 整合步驟（6 個 agent 全部完成後）
1. 讀取 `output/*.md`。
2. 回填 [`../itinerary.tsv`](../itinerary.tsv)：每格取代 `〔YT：待研究填入〕` 為**單行緊湊式**「點名｜頻道+短連結｜來源等級A/B/C｜最近站(電梯)｜室內/戶外+雨備」；完整佐證放 `FINAL_GUIDE.md` 用編號對應，避免可追溯資訊在 tsv 流失。
3. 彙整成 `output/FINAL_GUIDE.md`（含：兩天行程表、各點 YouTube 出處＋等級、交通帶推車說明、**兩層雨備庫**、美食清單、**撤退/放棄速查**、**颱風應變頁**、費用、行前與出發前查證清單）。
4. 產生 `output/travel-guide.html`（mobile-first 離線單頁，手機可開、Ctrl+P 列印 PDF；含每點 YouTube 連結、撤退速查、颱風頁）。
5. **逐條 checkbox 複查**：
   - □ 每點有 YouTube 出處、**已實際 WebFetch 開啟驗證為真實且主題相符**、並標 A/B/C 來源等級
   - □ 每個戶外點有室內雨備（含撤退途中防護）
   - □ 全文不得出現計程車/自駕/包車字樣（例外須標註）
   - □ 推車無障礙與電梯已標
   - □ 13:00-15:30 午睡時段有具體安排（含**抵達日**與**第二天高鐵上補眠**）
   - □ 每點主選＋備案齊全；高風險點有「整段替代」
   - □ 有撤退/最短收工速查
   - □ 有颱風應變頁
   - □ 2歲票價/可否搭乘已列待查證
   - □ 費用表逐項標明 2 歲幼兒是否計費（與 Agent 5 一致）
