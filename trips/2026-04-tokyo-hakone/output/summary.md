## 1. 天氣預報與穿著建議

### 東京（4/10-4/12、4/14-4/15）

| 項目 | 資訊 |
|------|------|
| 平均氣溫 | 白天約 15–20°C，早晚約 10–13°C |
| 降雨機率 | 四月中旬偶有春雨，建議準備折疊雨傘 |
| 天氣特色 | 春季櫻花末期，天氣宜人但早晚偏涼 |

### 箱根（4/12-4/14）

| 項目 | 資訊 |
|------|------|
| 平均氣溫 | 白天約 12–16°C，早晚約 7–10°C（山區較涼） |
| 降雨機率 | 山區霧氣較重，4月降雨較頻繁 |
| 天氣特色 | 大涌谷海拔約 1,000m，溫差大、風大 |

### 穿著建議

- **基本層**：薄長袖或棉質T恤
- **中間層**：薄毛衣或刷毛外套
- **外層**：防風防雨薄外套（必備，尤其箱根山區）
- **鞋子**：防滑平底鞋或運動鞋，避免高跟鞋（石板路多）
- **2歲半幼兒**：多帶一件備用外套及替換衣物，戶外活動注意防寒
- **60歲長輩**：早晚建議圍巾或薄頸套，關節保暖為優先
- **雨具**：每人一把折疊傘，或準備輕量雨衣（推車可加雨罩）

## 3. 大涌谷空氣品質與安全評估

### 大涌谷（大涌谷）概況

- 位置：箱根山，海拔約 1,044 m
- 特色：活火山硫磺蒸氣噴發區，黑玉子（黑蛋）著名景點

### 空氣品質說明

| 項目 | 說明 |
|------|------|
| 主要污染物 | 二氧化硫（SO₂）、硫化氫（H₂S） |
| 正常狀態 AQI | 約 50-80（輕度污染，主要為天然火山氣體） |
| 硫磺濃度 | 隨風向與火山活動強度變化極大 |
| 官方監測 | 箱根山火山活動觀測系統 24 小時監測 |

### 家庭旅客安全建議

| 旅客 | 風險等級 | 建議 |
|------|----------|------|
| 2歲半幼兒 | 高風險 | 硫磺氣體對嬰幼兒呼吸道刺激性強，**建議縮短停留時間至 30 分鐘以內** |
| 60歲長輩 | 中等風險 | 呼吸系統較弱者應注意，有氣喘者**建議不進入核心區域** |
| 一般成人 | 低風險 | 正常狀態下短暫停留無礙 |

### 注意事項

- 訪問前查詢當天火山活動警戒等級：[箱根山火山情報](https://www.jma.go.jp/bosai/volcano/#area_type=volcano&area_code=05&id=505&lang=ja)
- 警戒等級 1（正常）= 可進入；警戒等級 2 以上 = 建議取消
- 推車通行：大涌谷觀光設施園區有鋪面步道，**推車可通行部分區域**，但蒸氣區域地面不規則，請抱起幼兒
- 強烈建議：若幼兒或長輩有呼吸道疾病，以大涌谷纜車站外觀景台遠觀即可，不必深入蒸氣區

## 6. 行前準備清單

### 餐廳預約（建議出發前 2-4 週）

- [ ] **Day1 晚餐**：上野/根岸附近素食友善餐廳（精進料理或有蔬食選項的日式料理）
- [ ] **Day2 午餐**：川越在地午餐（うなぎ鰻魚或蕎麥麵，確認素食選項）
- [ ] **Day3 晚餐**：箱根懷石料理（告知1位全素食旅客，提前溝通）
- [ ] **Day4 午餐**：強羅/彫刻の森美術館附近餐廳
- [ ] **Day5 晚餐**：淺草/押上附近（隅田川旁景觀餐廳）
- [ ] **Day6 午餐**：返台前機場附近或輕食

### 票券購買

- [ ] **Skyliner 來回票**：京成電鐵官網 / 台灣代理商預購（省時省錢）
- [ ] **箱根フリーパス 2日券**：小田急電鐵官網或台灣旅行社
- [ ] **東京スカイツリー 展望台**：官網預購可省排隊時間

### IC 卡 / 支付準備

- [ ] 每人準備 **Suica** 或 **PASMO**（建議出發前在台灣或抵日後機場購買）

- [ ] 準備足夠現金（部分小餐廳、神社只收現金）
- [ ] 換匯：每人建議帶 ¥30,000-50,000 現金備用

#### 素食說明日文卡（建議列印攜帶）
```
私は完全菜食主義者（ベジタリアン）です。
以下のものが含まれていないお料理をお願いします：
- 肉（牛・豚・鶏・羊など）
- 魚・海鮮（だし・魚醤含む）
- 卵
- 乳製品
ありがとうございます。
```
（譯：我是全素食者。請提供不含肉類、魚/海鮮（包含高湯、魚醬）、蛋、乳製品的料理。謝謝。）

- [ ] 防曬乳（春季紫外線強）
- [ ] 口罩（大涌谷硫磺區建議佩戴）
