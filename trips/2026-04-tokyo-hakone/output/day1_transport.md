# Day 1 交通路線：台灣 → 成田機場 → 上野 → 飯店
## 2026/4/10（週五）

---

## 旅客資訊
- 人數：8人（60歲長輩 2 位、30多歲成人 4 位、6歲兒童 1 位、2歲半幼兒 1 位）
- 特殊需求：1 位長輩全素食、1 位幼兒需推車
- 住宿：嵐ホテル（東京都台東区根岸 2-16-5）

---

## 整體路線概覽

```
台灣桃園機場 → 飛行約 3.5 小時 → 成田機場第一航廈 (NRT T1)
     ↓
成田機場第一航廈 B1F → 京成 Skyliner → 京成上野站（約 41 分鐘）
     ↓
京成上野站 → 步行 → 嵐ホテル（約 20 分鐘 / 1.2 km）
```

## 第一段：成田機場第一航廈 → 京成上野站

```
搭乘 Skyliner（スカイライナー）
├─ 移動方式：京成 Skyliner 特急列車
├─ 出發站：成田機場第一航廈站（B1F）
├─ 抵達站：京成上野站
├─ 所需時間：約 41 分鐘
├─ 班距：約每 20-40 分鐘一班（尖峰時段）
├─ 費用：
│   ├─ 成人（12歲以上）：單程 ¥2,570 / 來回 ¥4,500
│   ├─ 兒童（6-11歲）：單程 ¥1,290 / 來回 ¥2,250（6歲兒童適用）
│   └─ 幼兒（未滿6歲）：免費（2歲半幼兒免費，無需購票）
├─ 購票地點：
│   ├─ 主要窗口：成田機場第一航廈 B1F 入境大廳出口附近
│   │   「京成 Skyliner & 交通服務中心」（京成スカイライナー & トラフィックサービス）
│   ├─ 自動售票機：B1F 京成線月台前
│   ├─ 網路預購：keisei.co.jp（可指定座位，建議提前預約）
│   └─ 注意：Skyliner 為對號入座，建議提前購票
├─ 推車友善度：⭕（全程可推車）
├─ 電梯位置：
│   ├─ 第一航廈 B1F：入境大廳→地下月台有電梯（近 7 號閘口方向）
│   ├─ 售票機旁設有無障礙服務台，可請站員協助
│   └─ Skyliner 車廂：每列車均設有大型行李架與無障礙座位
├─ 長輩注意：
│   ├─ 座位舒適，有靠背，適合長途乘坐
│   ├─ 行李可放車廂端頭的行李架（免費）
│   └─ 若攜帶大型行李（3邊合計 160cm 以上），建議使用「Skyliner 行李置放服務」
└─ 備註：
    ├─ 共 8 人購票：成人 6 張 × ¥2,570 = ¥15,420
    │              兒童 1 張 × ¥1,290 = ¥1,290
    │              幼兒 1 名 = ¥0（免費）
    │              小計：¥16,710（單程）
    ├─ 來回票：成人 6 × ¥4,500 + 兒童 1 × ¥2,250 = ¥29,250

```

├─ 移動方式：步行
├─ 距離：約 1.2 km
├─ 所需時間：約 15-20 分鐘（輕步行）
│           約 25-30 分鐘（推車 + 行李）
├─ 費用：免費
├─ 推車友善度：⚠️（大致可推車，但部分路段有坡度）

│   ├─ 京成上野站出口：主要出口為 3 號出口（地面層，無需電梯）
│   └─ 沿途為一般街道，無障礙設施普通
├─ 步行路線說明：
│   ├─ 出京成上野站 3 號出口
│   ├─ 往不忍通り（不忍通）方向
│   ├─ 沿不忍通り往北（往根岸方向）步行約 800m
│   ├─ 右轉進入根岸 2 丁目
│   └─ 嵐ホテル（台東区根岸 2-16-5）抵達
├─ 高度變化（海拔）：
│   ├─ 京成上野站出口：約 5m（地面層）
│   ├─ 沿不忍通り：平坦，高差不超過 5m
│   └─ 整體坡度：平緩，推車負擔輕
├─ 長輩注意：
│   ├─ 路程約 20 分鐘，建議長輩可搭計程車
│   ├─ 長途飛行後攜帶行李，建議全體搭計程車（約 ¥800-1,000）
│   └─ 計程車乘車處：京成上野站正門前廣場

    ├─ 若攜帶大型行李，強烈建議搭計程車（8人可叫2台）

    └─ 可在站前計程車招呼站排隊，或使用 Japan Taxi App 叫車

## 行前準備建議

1. **提前購票**：Skyliner 建議在台灣出發前購買（Klook、KKday 或京成官網），可指定座位，確保 8 人相鄰
2. **座位建議**：購票時選擇同一車廂，方便照顧幼兒和長輩
3. **行李處理**：超大型行李可利用「宅急便」服務（機場→飯店），提前一天安排，費用約 ¥1,500-2,000/件
4. **幼兒推車**：Skyliner 各車廂端頭有空間可放推車（不折疊），請向車站工作人員確認
5. **素食長輩**：機場 B1F 及上野站附近有便利商店，可購買素食飯糰或蔬菜類食物
