```

├─ 購票：不需購票（已有箱根周遊券）
├─ 推車友善度：⭕ 可行
├─ 電梯位置：此段為步行道路，無需電梯
├─ 長輩注意：二之平到彫刻の森駅距離約350公尺，路面平緩，但需穿越
│            縣道730號線（注意車輛）
└─ 備註：早上出發建議 09:00 前，避免人潮

### 第二段：彫刻の森駅 → 強羅駅（箱根登山鐵道）

```
箱根登山鐵道 彫刻の森 → 強羅
├─ 移動方式：箱根登山鐵道（Hakone Tozan Railway）
├─ 所需時間：約 3 分鐘（1站）
├─ 費用：箱根周遊券涵蓋（成人單程 ¥170，兒童 ¥90）
├─ 購票：出示箱根周遊券即可直接乘車
├─ 推車友善度：⭕ 可推車上車（車廂寬敞，推車可放車廂連接處）
├─ 電梯位置：強羅駅有電梯（位於站內主要出入口旁）
├─ 長輩注意：登山電車坡度大（最大80‰），起步/停車時注意抓握手把
│            座位較少，優先讓長輩就座
└─ 備註：班次頻繁，約 15～20 分鐘一班
         強羅站月台有坡度，下車時注意腳步
         箱根周遊券請在乘車前備妥（8人建議分2組確認票券）
```

```
箱根登山鋼索鐵道 強羅 → 早雲山

├─ 所需時間：約 10 分鐘（全程 5 站）

├─ 購票：出示箱根周遊券直接搭乘

│   - 建議將幼兒抱起，推車折疊後立放於車廂側邊

├─ 電梯位置：強羅站：電梯可達月台（靠公園下方向）
│            早雲山站：有升降設備通往空中纜車月台

## 大涌谷纜車停駛替代方案

> 大涌谷纜車（空中纜車）因火山活動或惡劣天氣停駛機率較高，請特別注意以下替代方案。

### 停駛可能原因
- 大涌谷火山活動警戒升級（最常見）
- 強風（風速超過 15 m/s）
- 大雨/濃霧

### 替代方案 A：全程改巴士（推薦）

```
替代路線 A：登山電車 + 巴士環線
強羅駅
 ↓ 箱根登山巴士（箱根フリーパス可用）
元箱根港
 ↓ 海賊船
箱根町港
 ↓ 巴士
強羅/彫刻の森

優點：完全陸路，不受天氣影響
缺點：無法遊覽大涌谷
時間：強羅→元箱根 約 40 分鐘
費用：箱根周遊券涵蓋
```
