# 東京箱根家族旅遊 完整規劃提示詞（MCP Google Maps）

## 使用方式
- **方式A**：將下方「完整版提示詞」一次貼入 Claude CLI
- **方式B**：到 `prompts/day1.md` ~ `day6.md` 逐天執行（推薦，避免 context 爆炸）
- 前提：Claude CLI 已設定好 `.mcp.json`（google-maps MCP）

---

## 完整版提示詞

```
你是一位專業的日本家庭旅遊規劃師。我有一個 2026/4/10-4/15 的東京＋箱根家族旅遊行程。
請使用 MCP Google Maps 所有可用工具（特別是複合工具）來完成以下所有任務。

重要：請大量使用以下工具，不要只靠文字回覆：
- maps_plan_route → 規劃每天的多站路線（最多25站）
- maps_compare_places → 並排比較每餐的主選vs備案餐廳
- maps_explore_area → 探索每個停留區域周邊吃喝玩樂
- maps_static_map → 為每天產生路線地圖圖片（含標記點）
- maps_directions → 每段交通的詳細步驟
- maps_place_details → 每個景點/餐廳的詳細資訊
- maps_search_nearby → 搜尋備案餐廳/便利設施
- maps_distance_matrix → 多點距離矩陣（購物區逛街順序）
- maps_weather → 查詢4月中旬東京/箱根天氣
- maps_elevation → 查詢箱根各站點海拔（評估長輩體力負擔）
- maps_search_along_route → 沿途找便利商店/藥妝店/尿布購買點
```
