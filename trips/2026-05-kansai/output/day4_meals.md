# 5/16（六）京都嵐山一日遊 三餐規劃

> 旅客：9人（成人6 + 小學生1 + 幼稚園1 + 幼幼班推車1）
> 特殊需求：1位全素長輩、推車1台
> 行程：早上大阪難波出發 → 嵐山 → 金閣寺 → 晚上回難波

---

## 早餐 07:00（難波飯店附近）

🍽️ つるまる饂飩 難波店（烏龍麵）【主選】
├─ 地址：大阪市中央区道頓堀1-7-22（道頓堀商圈）
├─ 營業時間：07:00–23:00（全年無休）
├─ 評分：⭐ 3.8 / 5.0
├─ 素食選項：✅ 可點「かけうどん」（單純昆布高湯烏龍麵）；需告知服務員不要柴魚高湯，改用昆布高湯版本。天婦羅類多含魚蝦，長輩宜點純麵
├─ 兒童：⭕ 無專屬兒童椅，但可自備或要求一般椅子；推車可摺疊後入座位旁
├─ 預約：不需預約，現場直接點餐（自動販賣機購票）
├─ 人均：¥400–700（烏龍麵 + 小菜）
├─ 9人可進：⭕ 連鎖大型店面，翻桌快，7:00 人潮少
└─ 注意：高湯底料為柴魚，素食長輩需特別告知要昆布高湯（konbu dashi）；可先準備日文小卡「昆布だしのみ、動物性食品不使用でお願いします」
