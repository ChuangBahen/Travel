# Day6 交通規劃｜2026/5/18（週一）退房 → KIX → 台灣

## 旅客組成
- 成人 6 人 / 小學生 1 人 / 幼稚園 1 人 / 幼幼班 1 人（推車）
- 出發地：大阪難波（Osaka Namba）住宿飯店
- 目的地：關西國際機場（KIX）T1 國際線
- 回程班機：午後出發（14:00 或 16:00 KIX → 台灣）

---

## 路線比較：難波 → KIX

### 方案 A｜南海ラピート（Rapi:t）特急【推薦首選】

```
🚃 難波 → 關西空港（南海ラピート特急）
├─ 路線：南海電鉄 難波駅 → 関西空港駅（直達）
├─ 所需時間：約 38 分鐘
├─ 費用：
│   成人 ¥1,490 / 小學生 ¥750（半價）
│   幼稚園 ¥750（小學生票）/ 幼幼班 免費（3歲以下）
│   9 人合計：成人6×¥1,490 + 小學生1×¥750 + 幼稚園1×¥750 = ¥10,440
├─ 班次：每小時約 2 班（整點/半點）
├─ 推車+行李：⭕ 車廂寬敞，有指定席，行李空間充足
├─ 購票：難波駅 南海電鉄 窗口（建議團體一起買）
│   或南海電鉄官網預訂（可選座位）
└─ 備註：藍色鋼鐵人外型特急，全車對號座，安靜舒適
         9人建議在窗口說明人數，詢問團體優惠
         ICOCA 不可搭乘（需另購紙票或指定席券）
```

### 方案 B｜南海空港急行【預算首選】

```
🚃 難波 → 關西空港（南海空港急行）
├─ 路線：南海電鉄 難波駅 → 関西空港駅（直達）
├─ 所需時間：約 45～50 分鐘
├─ 費用：
│   成人 ¥970 / 小學生 ¥490（半價）
│   幼稚園 ¥490 / 幼幼班 免費（3歲以下）
│   9 人合計：成人6×¥970 + 小學生1×¥490 + 幼稚園1×¥490 = ¥6,800
│   相比 Rapi:t 省下約 ¥3,640
├─ 班次：每小時 2～3 班（較密集）
├─ 推車+行李：⚠️ 自由席，上下班高峰可能擁擠，行李較多時需占位
├─ 購票：自動售票機 / ICOCA 可刷（幼稚園以下刷大人卡免費）
└─ 備註：一般車廂，與 Rapi:t 同月台，看清楚指示牌上車
         週一早上上班人潮較多，9人+推車+行李建議避開 8:00-9:00
```

### 方案 C｜JR 関空快速【不推薦】

```
🚃 難波 → 天王寺 → 関西空港（JR 関空快速）
├─ 路線：JR 難波駅（徒歩 10 分）→ 天王寺 → 関西空港
├─ 所需時間：約 65～75 分鐘（含換車）
├─ 費用：成人 ¥1,080 / 小學生 ¥540
├─ 推車+行李：⚠️ 需轉車，不方便
└─ 備註：❌ 不建議，JR 難波駅與南海難波距離遠（需步行換站）
         時間長且不直達，9人+推車+大型行李轉車麻煩
         JR Pass 可使用，但行程已到最後一天通常已不值得
```

## 推薦結論

| 考量 | 推薦方案 |
|------|---------|
| 舒適度最高（有座位保證） | 方案 A：ラピート特急 |
| 省錢（省 ¥3,640） | 方案 B：空港急行 |
| 有嬰幼兒推車+大行李 | 方案 A 更優（指定席空間確定） |
| 週一退房高峰 | 方案 A 最安心 |

**最終建議：方案 A（ラピート特急）**
9 人 + 推車 + 9 件大行李 + 週一退房，確保有位置是最優先考量。
指定席預訂後全員座位確定，不必擔心行李和推車的空間問題。

## 步行路線：飯店 → 南海難波駅

```
🚶 飯店（大阪難波周邊） → 南海電鉄 難波駅
├─ 路線：推車 + 行李推車路線
│   飯店大廳 → 出門右轉/左轉至御堂筋 → 沿難波駅指示牌
│   → 南海難波駅 南口（1F 地面層入口）
├─ 所需時間：徒步 5～15 分鐘（依飯店位置而異）
├─ 高度差：平坦（難波周邊地勢平坦）
├─ 推車友善度：⭕ 南海難波駅有電梯直達各月台
├─   電梯位置：南口入口旁、中央大廳旁均有電梯
├─   行李寄放：南海難波駅 1F 有 COIN LOCKERS（退房後先寄）
├─ 長輩注意：若行李多，建議預叫計程車（MK 計程車 2 台＝9人）
│           難波→KIX 計程車約 ¥15,000-¥20,000（含高速費）
└─ 備註：飯店可協助叫車或確認電梯路線
```

## 關西機場 T1 到達後流程

### 抵達 KIX 後步驟

```
🛬 KIX 到達 → 登機
├─ 步驟 1：下車 → 入境（出發層 4F/3F）
│   関西空港駅 → T1 連接天橋（有電梯）→ 出境大廳
├─ 步驟 2：ICOCA 押金退還
│   ⭕ 地點：T1 二樓（2F）到着口方向 みどりの窓口（JR 服務中心）
│   ⭕ 或 T1 一樓（1F）自動精算機（部分機台可退）
│   押金：每張 ¥500，9人最多退 ¥4,500（若全員有卡）
│   注意：卡內餘額也可退還，但需排隊，預留 15 分鐘
├─ 步驟 3：辦理登機（Check-in）
│   ⭕ 地點：T1 三樓（3F）出發大廳 → 找您的航空公司櫃台
│   台灣主要航空：中華航空（CI）/ 長榮航空（BR）/ 台灣虎航（IT）
│   所需時間：9人推薦預留 30～45 分鐘
│   推車寄艙：在 Check-in 櫃台即可辦理，嬰兒推車免費寄艙
│             ※ 建議寄艙前拍照記錄推車狀態
├─ 步驟 4：安全檢查 + 出境審查
│   所需時間：預留 30 分鐘
│   推車：過安檢前寄艙，或使用機場輪椅
│   長輩：安全人員可協助，不必慌張
├─ 步驟 5：免稅購物（出境後）
│   地點：出境後 2F 免稅店區
│   推薦：Royce 生巧克力、白い恋人（北海道）、神戶布丁
│         蛋捲（ルーベラ）、抹茶點心、威士忌（山崎/響）
│   素食注意：大部分甜點含奶/蛋，嚴格素食者確認成分
└─ 步驟 6：登機門候機
    KIX T1 國際線 步行時間：出境後到登機門約 10～20 分鐘
    建議班機前 45 分鐘抵達登機門
```

## ICOCA 卡退卡指南

```
💳 ICOCA 押金退還
├─ 地點：關西機場（KIX）T1 二樓（2F）
│   → JR 西日本みどりの窓口（綠色服務窗口）
│   → 或 JR 自動精算機（部分機台支援退卡）
├─ 退還金額：押金 ¥500 + 卡內剩餘金額（扣手續費 ¥220）
│   例：卡內剩 ¥300 → 退還 ¥500+¥300-¥220 = ¥580
├─ 需要攜帶：ICOCA 卡本身（不需護照）
├─ 等待時間：約 5～10 分鐘/人，9人預留 20～30 分鐘
├─ 注意：
│   - 記名 ICOCA 與無記名 ICOCA 都可退
│   - 卡內餘額不足 ¥220 手續費時，押金退 ¥500，餘額放棄
│   - 退卡後當場確認金額
└─ 替代方案：餘額少的卡留著當紀念品或下次使用
```
