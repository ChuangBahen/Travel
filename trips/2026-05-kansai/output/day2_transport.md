# Day2 交通規劃：神戶一日遊
**日期：** 2026/5/14（四）  
**人員：** 9 人（成人 6 + 小學生 1 + 幼稚園 1 + 幼幼班（推車）1）  
**主題：** 北野異人館 → 神戶港（Meriken Park）→ 麵包超人博物館  
**住宿基地：** 大阪難波  

---

## 全日行程總覽

```
難波出發（8:30）
  ↓ 阪神なんば線直通特急（40分）
神戶三宮（9:10）
  ↓ 計程車上山（10分 ¥1,000前後）← 強烈建議！推車+長輩
北野異人館街（9:20）
  ↓ 步行下山（25分・沿北野坂）← 下坡推車可行
神戶港 Meriken Park（11:00）
  ↓ 步行（10分）
神戶麵包超人博物館・umie モザイク（11:30〜14:30）
  ↓ 步行 or 計程車（15分 or ¥800）
神戶三宮（15:00）
  ↓ 阪神なんば線直通特急（40分）
難波返回（15:40）
```

```
阪神なんば線直通特急
├─ 路線：阪神なんば線 難波駅 → 神戶三宮駅（直通、不換車）
├─ 票價：
│    成人    ¥410
│    小學生  ¥210（6歲以上）
│    幼稚園  ¥210（要購票，未滿6歲免費條件：大人同行1名帶1名）
│    幼幼班  ¥0〜¥210（滿1〜未滿6歲，每位大人可免費帶1名）
│    9人合計概算 ¥2,460〜¥2,870（視幼兒免費計算）
├─ 時間：約 40 分鐘（途中停靠主要站）
├─ 發車頻率：日間約每 12〜15 分 1 班
├─ 推車友善：⭕
│    ・難波駅（阪神）設有電梯，月台有點空間

     ・9時前搭車可避開通勤尖峰
```

### 段 2：神戶三宮 → 北野異人館街

#### 方案 A（強烈建議）：計程車上山

```
三宮駅 → 北野異人館街（計程車）
├─ 路線：三宮駅東口 計程車乘り場 → 北野坂頂部（風見鶏の館附近）
├─ 票價：
│    一般計程車（4人座）：¥800〜¥1,200 / 台
│    9人 → 需 2〜3 台（小型計程車2台 or 大型1台+小型1台）
│    合計概算：¥1,600〜¥2,400
├─ 時間：約 8〜12 分
├─ 推車友善：⭕
│    ・計程車直接抵達異人館坡頂，完全省去上坡
│    ・推車可折疊放入行李廂
├─ 電梯：不需要
├─ 海拔變化：三宮（海拔約 10m）→ 北野異人館（海拔約 80〜100m）
│    高度差約 80m、坡度約 10〜15%（徒步上坡約 20〜25 分、推車極辛苦）
└─ 備註（9人團）：
     ・三宮駅東口有計程車排班，搭2台即可
     ・告知司機「北野異人館、風見鶏の館附近」即可
     ・建議上山搭車、參觀後步行下坡（下坡推車舒適，沿途風景好）
     ・大型計程車（ジャンボタクシー）可載7〜9人，提前電話預約更安心
       神戶タクシー：078-861-1212
```

#### 方案 B（備用）：步行上坡（不建議 9 人推車組）

```
三宮駅 → 北野異人館街（步行）
├─ 路線：三宮駅 → 北野坂通り → 北野異人館街
├─ 時間：約 20〜25 分（含上坡）
├─ 推車友善：⚠️
│    ・坡度明顯（約 10%）、部分路段無坡道
│    ・長輩體力消耗大，不建議當天第一段就徒步上坡
├─ 海拔變化：+80〜100m（上坡全程）
└─ 備註：若選步行，建議在三宮先享用早餐補充體力
```

#### 方案 C（備用）：City Loop 巴士（上山站）

```
三宮（City Loop 巴士）→ 北野異人館（16番站）
├─ 路線：三宮（バスターミナル）→ 北野異人館 City Loop 16號站
├─ 票價：
│    成人 1次 ¥260 / 一日券 ¥700
│    小學生 1次 ¥130 / 一日券 ¥350
│    幼兒 免費（保護者同伴）
│    9人 1次概算 ¥1,820（成人6×¥260 + 小學生¥130 + 幼稚園¥130）
├─ 時間：約 15〜20 分（班距 20〜30 分，須等車）
├─ 推車友善：⚠️
│    ・需折疊推車上車、車廂空間有限
│    ・9人同時上車可能需分批（視巴士空間）
└─ 備註：一日券划算但班次少，9人使用計程車更省時
```

### 段 3：北野異人館 → 神戶港 Meriken Park（步行下山）

```
北野異人館街 → 神戶港 Meriken Park（步行）
├─ 路線：北野異人館 → 北野坂 → 三宮 → 元町 → 海岸通り → Meriken Park
├─ 時間：約 30〜40 分（含下坡 + 市區步行）
├─ 推車友善：⭕
│    ・下坡坡道，推車容易操控
│    ・海岸通り至 Meriken Park 段為平地
├─ 電梯：市區道路，不需電梯
├─ 海拔變化：-80〜100m（持續下坡至海拔接近 0m）
└─ 備註（9人團）：
     ・下坡路段風景優美，沿途可看異國風建築
     ・元町商店街可稍作休息/購物（有頂蓋、推車友善）
     ・若長輩腳力不佳，可在三宮搭計程車至 Meriken Park（¥800前後）
     ・備選路線：步行至元町駅 → JR 神戶駅（2分 ¥200）→ 步行至 Meriken Park（15分）
```

## 9 人團特別建議

1. **計程車策略**：三宮→北野務必搭計程車（2台），保留長輩與幼兒體力給景點；回程也可安排計程車，不強求步行。

2. **時間配置建議**：
   - 北野異人館：90〜120 分（建議主攻風見鶏の館外觀 + ラインの館免費館）
   - Meriken Park + 港塔：60〜90 分
   - 麵包超人博物館：120〜180 分（孩子們的重點！）

3. **午餐選擇**：
   - 北野異人館街：多家西式、法式餐廳，部分可容納 9 人大桌（建議提前預約）
   - Meriken Park 附近：神戶牛排館、海景餐廳
   - umie モザイク：餐廳選擇最多，無需預約

4. **City Loop 巴士一日券**：若不搭計程車、行程輕鬆，成人 ¥700 可無限搭乘（連接三宮、北野、港區），但班次少（20〜30 分 1 班），9 人團搭計程車更有效率。

5. **阪神一日乘車券**：難波→三宮往返已需 ¥820/人，若有意多搭阪神線，可考慮「阪神1日乗車券」¥1,000（成人），但本行程往返 2 次即已划算。

6. **幼兒免費乘車規則**：
   - 阪神電車：6歲未滿幼兒，每位大人可攜帶 2 名免費（座席不佔用）
   - 幼稚園生（6歲、就學前）：依日本規定，未滿6歲免費；已入學小學生則須購票
   - 請依 2026 年 5 月實際年齡確認孩子票種
