## ⚠️ 幼幼班與幼稚園小朋友 鹿群安全重要提醒

```
鹿群兒童安全注意事項（非常重要！）
├─ 危險行為 1：手持鹿仙貝引來大量鹿包圍
│   → 奈良鹿嗅到食物會從四面八方衝過來，小孩容易被撞倒
│   → 對策：由大人拿仙貝，小孩站在大人背後或抱起，再一起餵
│
├─ 危險行為 2：鹿低頭搶食時頭頂會頂到小孩臉部/胸部
│   → 幼稚園（約 100cm）臉部與鹿頭同高，風險最高
│   → 對策：讓孩子蹲下或由大人抱著，由上方餵鹿
│
├─ 危險行為 3：鹿咬住衣物/書包不放（誤以為是食物）
│   → 對策：避免穿著淺色、咖啡色衣物（鹿仙貝顏色），不掛零食包
│
├─ 危險行為 4：推車在鹿群中被舔/撞倒
│   → 鹿會舔推車的餐盤托盤、袋子，甚至將頭伸入推車
│   → 對策：
│       ✅ 推車上不放食物或零食袋
│       ✅ 幼幼班抱起後再進入鹿群聚集區餵鹿
│       ✅ 一人專責看推車，維持車身穩定
│       ✅ 推車置於鹿群外圍，大人圍繞形成保護圈
│
├─ 危險行為 5：食物吃完後鹿仍追著跑（鹿會做「低頭敬禮」要食物）
│   → 對策：食物吃完後，兩手攤開向鹿展示「沒有了」，大部分鹿會離開
│            若仍追趕，立即走向人多處或店舖内
│
└─ 奈良鹿是「國寶級野生動物」（天然紀念物），不可推打鹿
   但若鹿頂撞你，可用手輕推鹿頭使其退開
```

## 奈良午餐推薦（全素食友善）

```
奈良午餐選擇（考量家族長輩全素食需求）
├─ 推薦 1：夢風ひろば（東大寺前）
│   - 位置：東大寺大佛殿正前方，奈良公園内
│   - 特色：集合多家餐廳的美食廣場
│   - 素食選項：有蕎麥麵店（問清醬汁不含柴魚），抹茶甜品
│   - 推車：⭕ 戶外廣場，推車方便
│
├─ 推薦 2：奈良町周邊（近鉄奈良駅南側）
│   - 有豆腐料理、精進料理（全素）専門店
│   - 「江戸川」等店有提供茶蕎麥麵（需確認湯底）
│   - 步行至奈良駅約 10 分鐘
│
├─ 注意事項（素食長輩）：
│   - 日本蕎麥麵高湯通常含「柴魚（かつおだし）」
│   - 必須明確說「出汁なし」或「精進だし」
│   - 豆腐料理店較常有純植物性選項
│
└─ 備用：便利店 7-11 / FamilyMart（近鉄奈良駅商業棟内）
    有飯糰・沙拉・素食麵包，緊急時可用
```
