# 京阪神家族旅遊 Day1 交通規劃
**日期**：2026/5/13（三）  
**人員**：9 人（成人 6 位、小學生 1 位、幼稚園生 1 位、幼幼班幼兒 1 位）  
**特殊需求**：1 名幼幼班幼兒使用推車  
**目的地飯店**：大阪難波 OMO7 大阪（難波站周邊 500m 內）

## 路線一覽：KIX → 難波（三條比較）

### 路線 A：南海ラピート（Rapi:t）特急

```
🚃 關西機場站 → 南海難波站（Rapi:t 特急）
├─ 車站/出口：2F 南海售票中心購票，月台 1・2 番線搭乘
├─ 票價：
│   成人 ¥1,490（含特急指定席）
│   小學生 ¥750（兒童半價+特急券）
│   幼稚園生 ¥750（同上）
│   幼幼班幼兒（未滿 6 歲）¥0（免票，不佔座位）
│   ────────────────────────
│   9 人試算：成人6 × ¥1,490 + 兒童2 × ¥750 = ¥10,440
├─ 時間：約 38 分鐘（直達，不停中途站）
├─ 班次：每 30 分鐘一班（旺季加班）
├─ 推車友善：⭕
│   ∟ 指定席（Beta 型）每節車廂前方有行李架空間
│   ∟ 推車可折疊放行李架，或置於座位旁走道
│   ∟ 建議預訂 1 號車廂靠近車廂端的座位
├─ 電梯位置：關西機場站─月台電梯在剪票口左側；難波站─1號出口旁
├─ 長輩注意：有冷氣、固定座位，行程長輩不必站立
└─ 備註：
    ∟ 建議出發前在台灣用「南海電鐵官網」或 Klook 預購，
      可折扣至 ¥1,330（外國人優惠 e-ticket）
    ∟ 9 人需確保同節車廂；訂票時備註「9 人同行」
    ∟ 大型行李可置車廂端行李架，不佔座位
```

### 路線 B：南海空港急行

```
🚃 關西機場站 → 南海難波站（空港急行）
├─ 車站/出口：2F 南海自動售票機購票，月台 3・4 番線搭乘
├─ 票價：
│   成人 ¥970
│   小學生 ¥490（兒童半價）
│   幼稚園生 ¥490（同上）
│   幼幼班幼兒 ¥0（免票）
│   ────────────────────────
│   9 人試算：成人6 × ¥970 + 兒童2 × ¥490 = ¥6,800
├─ 時間：約 50 分鐘（途停天下茅ヶ崎、堺、住之江公園等）
├─ 班次：每 15～30 分鐘一班
├─ 推車友善：⚠️
│   ∟ 一般通勤電車（無指定席）
│   ∟ 尖峰時段（17:00-19:00）車廂擁擠，推車不易移動
│   ∟ 非尖峰可選車廂端或優先席旁空間停放推車
│   ∟ 9 人+行李+推車在一般車廂較為困難
├─ 電梯位置：同南海路線；難波站有電梯但需換月台
├─ 長輩注意：站立時間可能較長，對 60 歲長輩體力消耗較大
└─ 備註：
    ∟ 無需事先預訂，直接用 ICOCA 刷卡或購買車票
    ∟ 若行李少、時間彈性可考慮此方案節省費用
    ∟ 行李架空間有限，9 人大型行李建議事先寄送
```

### 路線 C：JR 関空快速（Haruka 快速）

```
🚃 關西機場站（JR） → 大阪城北詰（換車）→ 難波（大阪メトロ）
├─ 車站/出口：2F JR 售票機購票（或 ICOCA 刷卡），月台 JR 島式月台
├─ 票價：
│   成人 ¥1,210（JR 関空快速至天王寺，再轉大阪メトロ）
│   ∟ JR 関空→天王寺 ¥1,060 + 大阪メトロ天王寺→難波 ¥290
│   ∟ 合計 成人 ¥1,350（需在天王寺出閘再刷入）
│   小學生 ¥680（各路線兒童半價合計）
│   幼稚園生 ¥680（同上）
│   幼幼班幼兒 ¥0（免票）
│   ────────────────────────
│   9 人試算：成人6 × ¥1,350 + 兒童2 × ¥680 = ¥9,460
├─ 時間：約 70 分鐘（含轉車等待約 10 分鐘）
├─ 班次：每 30 分鐘一班
├─ 推車友善：⚠️
│   ∟ JR 快速車廂無指定席，推車需於車廂端擺放
│   ∟ 天王寺換乘大阪メトロ需出站換票，有電梯可使用
│   ∟ 推車過閘口需使用寬幅閘門
├─ 電梯位置：天王寺站 JR→メトロ換乘有電梯（B1F 連絡通道）
├─ 長輩注意：需換車一次，攜帶大型行李時體力消耗最大
└─ 備註：
    ∟ 不建議 9 人+行李+推車選此方案（換車次數多）
    ∟ 若持 JR Pass 可免費搭 Haruka 特急至天王寺
```

## 三路線總比較表

| 比較項目 | A：南海Rapi:t特急 | B：南海空港急行 | C：JR関空快速 |
|----------|:----------------:|:---------------:|:-------------:|
| 所需時間 | 38 分鐘 | 50 分鐘 | 70 分鐘（含轉車） |
| 成人票價 | ¥1,490 | ¥970 | ¥1,350 |
| 9 人總費用 | ¥10,440 | ¥6,800 | ¥9,460 |
| 直達難波 | ✅ | ✅ | ❌（需換車） |
| 指定座位 | ✅ | ❌ | ❌ |
| 推車友善 | ⭕ | ⚠️ | ⚠️ |
| 行李空間 | 充足 | 有限 | 有限 |
| 長輩舒適度 | ⭕ 最佳 | ⚠️ 尚可 | ❌ 不建議 |

**結論：9 人大家族（含推車、大型行李）強烈建議選擇 A：南海Rapi:t特急**

```

│   ∟ 難波站中央口旁電梯直達地面
│   ∟ 黑門市場方向：沿千日前通步行無台階
├─ 長輩注意：
│   ∟ 道頓堀橋（戎橋）周邊人潮密集，推車需謹慎

    ∟ 飯店 Check-in 通常 15:00 後
```

## KIX → 難波 路線海拔說明

| 路段 | 地形說明 |
|------|---------|
| 關西機場（KIX） | 海拔約 4-6m（人工島，填海造陸） |
| 连絡橋（Sky Gate Bridge R） | 海拔約 15-20m（跨越大阪灣，橋梁高架） |
| 泉佐野市區路段 | 海拔約 5-10m（大阪府泉南平原） |
| 堺市通過段 | 海拔約 3-8m |
| 難波（南海難波站） | 海拔約 5-7m |

整體路線地勢平坦，由海平面附近的人工島，跨越連絡橋後進入大阪平原，至難波均為低平地形，**無明顯坡度**，適合推車使用。

| **B：台中 TXG → 關西 KIX（長榮）** | 長榮 BR177（早班 08:30 起飛，約 12:00 抵達）| 長榮 BR178（午班 13:30 起飛）| 成人 NT$9,000-16,000 | 中部旅客、早到可多玩半天 |
| **B：台中 TXG → 關西 KIX（虎航）** | 虎航 IT284（早班 09:00 起飛）| 虎航 IT283（午班 12:30 起飛）| 成人 NT$3,500-7,000 | 中部旅客、預算最省 |

**路線選擇建議：**
- 北部旅客（台北、桃園、新竹）→ 選 **A 桃園出發**，班次較多選擇彈性大
- 中部旅客（台中、彰化、南投）→ 選 **B 台中出發**，免去北上交通，早班到關西可多一個下午觀光
- 9 人成本考量：台中虎航最省，但需注意行李限額（廉航通常 20kg/人需另購）

## 推車與無障礙重要提示

1. **關西機場 T1 入境大廳**：電梯位於大廳四角，2F 連絡橋也有電梯直達 JR/南海月台層
2. **南海 Rapi:t**：每節車廂前後端有行李放置區，推車可折疊立放（建議折疊後以車頭固定帶固定）
3. **難波站出口**：建議使用「中央改札口」旁的無障礙出口，有電梯直達地面
4. **道頓堀戎橋**：橋面平坦，但周邊人潮多，尖峰時間推車需小心
5. **黑門市場方向**：人行道寬敞，路面平整，適合推車
