## 第 1 段：難波 → 大阪港（海遊館）

```
大阪地下鐵中央線  難波 → 大阪港駅
├─ 移動方式：大阪メトロ 中央線（Chuo Line）
│             難波駅（C16）→ 本町駅（C16 換乘接 C11）→ 阿波座駅 → 大阪港駅（C11）
│             ※ 難波の中央線乗り場は南海・近鉄難波駅から地下道で接続
│             ※ 中央線は難波に直接入らない。正確には：
│               【難波近辺からの正確ルート】
│               近鉄難波/南海難波 → 大阪メトロ四つ橋線「なんば」駅 → 本町 → 中央線に乗換 → 大阪港
│               または近鉄なんば駅 → 徒歩 → 中央線コスモスクエア方面に乗車
│             ※ 最も簡単なルート：
│               大阪メトロ「なんば駅」→ 四つ橋線 or 御堂筋線 → 本町駅 → 中央線 → 大阪港
├─ 所需時間：約 30-35 分（本町での乗換含む）
├─ 費用：成人 ¥280 / 小學生 ¥140 / 幼稚園・幼幼班 ¥0（未就學免費）
│         9 人合計：6×¥280 + 1×¥140 = ¥1,820（片道）
├─ 購票：なんば駅 自動售票機 または ICOCA（IC カード）直接感應
│         ★ OSAKA 海遊きっぷ（套票）購入なら乗車不要（下記参照）
├─ 班次：中央線は約 4-7 分間隔（平日日中）
├─ 出發建議：09:30 難波発 → 10:00 大阪港着
├─ 推車友善度：⭕（大阪メトロ全駅バリアフリー対応）
├─ 電梯位置：
│   なんば駅：御堂筋線・四つ橋線各改札にエレベーター
│   本町駅：中央線乗換エリアにエレベーター
│   大阪港駅：改札～地上エレベーター完備 ✅
├─ 長輩注意：本町での乗換は同じ地下通路内、約 3 分歩くが迷わない
└─ 備注：9 人同時改札 → 改札機の前後で列を作り、スムーズに通過
```
