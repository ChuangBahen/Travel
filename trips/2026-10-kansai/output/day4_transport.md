
## 第 1 段：難波 → 嵐山（三路線比較）

### 【路線 A】阪急（推薦）

```
🚃 難波 → 阪急嵐山駅
├─ 移動方式：大阪メトロ堺筋線（天下茶屋方面）→ 天神橋筋六丁目 乗換
│            阪急京都本線（特急/準急）→ 桂 乗換
│            阪急嵐山線（各停）→ 嵐山
├─ 所需時間：約 65〜75 分鐘（含候車・換乘）
├─ 費用：大人 ¥570 × 6 = ¥3,420
│        小學生（6歳〜12歳）¥290 × 2 = ¥580
│        幼稚園（3〜5歳）¥290 × 1 = ¥290
│        幼兒（未滿3歳）無料 ¥0
│        9人合計：¥4,290
├─ 購票：ICOCA / PiTaPa 可直接刷（推薦）
│        難波駅 自動售票機亦可購硬票
├─ 推車友善度：⭕ 阪急各站設電梯
├─ 電梯位置：
│   難波（堺筋線）→ 改札附近有電梯
│   天神橋筋六丁目 → 乗換通路有電梯（留意指標）
│   桂駅 → 嵐山線ホーム有電梯
│   嵐山駅 → 改札出口有電梯
├─ 長輩注意：桂乗換月台需走天橋，有電梯但距離稍遠；
│            特急→桂→各停嵐山線，注意月台編號
└─ 備註：嵐山線班距 20 分鐘，桂乗換後需候車
         建議 09:00 前從難波出發（週五人潮相對少）
         阪急嵐山駅出口直通渡月橋（步行 3 分鐘）★最便利★
```

```
🚃 難波(JR難波) → 京都駅 → 嵯峨嵐山駅
├─ 移動方式：大和路快速→JR難波 → 京都（大阪駅経由、所要約65分）
│            JR嵯峨野線（山陰本線）→ 嵯峨嵐山（約15分）
├─ 所需時間：約 85〜95 分鐘（含京都乗換）
├─ 費用：大人 ¥1,100 × 6 = ¥6,600
│        小學生 ¥550 × 2 = ¥1,100
│        幼稚園 ¥550 × 1 = ¥550
│        幼兒 無料 ¥0
│        9人合計：¥8,250
├─ 購票：ICOCA 利用可；JR難波〜難波 乗換に注意
│        ※ JR難波は南海難波とは別駅（徒歩5分）
├─ 推車友善度：⭕ JR嵯峨嵐山駅にエレベーター完備
├─ 電梯位置：JR難波・大阪・京都・嵯峨嵐山各駅に設置
├─ 長輩注意：JR難波→大阪駅 区間は電車本数多く乗換しやすいが
│            難波から JR難波まで歩く距離あり（地下道経由5分）
└─ 備註：嵯峨嵐山駅〜渡月橋 徒歩 約10分
         費用が路線Aより約¥4,000高い（9人比較）
         JR西日本「嵐山」は嵯峨嵐山駅が正式名（混同注意）
```

### 【路線 C】近鉄 + 嵐電

```
🚃 近鉄難波 → 京都駅 → 嵐電嵐山本線 → 嵐山駅（四条大宮経由）
├─ 移動方式：近鉄特急・急行 → 大和西大寺 → 京都（所要 約45〜60分）
│            地下鉄東西線・バス → 四条大宮 乗換（所要 約20分）
│            嵐電嵐山本線 → 嵐山（所要 約20分）
├─ 所需時間：約 100〜120 分鐘（乗換多・迴圈較複雑）
├─ 費用：近鉄（難波〜京都）大人 ¥1,160 × 6 = ¥6,960
│        小學生 ¥580 × 2 = ¥1,160
│        幼稚園 ¥580 × 1 = ¥580
│        地鉄+嵐電 大人 ¥500 × 6 = ¥3,000 程度
│        9人合計：¥12,000〜¥13,000（概算）
├─ 推車友善度：⚠️ 嵐電は旧型車両も多く、ステップあり
├─ 電梯位置：近鉄京都・四条大宮 周辺は限定的
├─ 長輩注意：乗換3回以上、距離が長い、推奨しない
└─ 備註：嵐電嵐山駅は天龍寺・竹林に最も近い（800m圏内）
         ただし全行程費用・時間とも不利
```

## 第 3 段：嵐山 → 金閣寺

### 【路線 X】嵐電 + 市バス 101/102（推薦）

```
🚃 嵐電嵐山駅 → 北野白梅町駅 → [市バス] → 金閣寺道バス停
├─ 移動方式：嵐電嵐山本線（各停）→ 北野白梅町（終点）
│            市バス 101号・102号・204号 → 金閣寺道
├─ 所需時間：嵐電 約 20 分 + バス 約 10 分 = 合計 約 45 分（待ち含む）
├─ 費用：嵐電 大人 ¥250 × 6 = ¥1,500
│        嵐電 小中学生 ¥130 × 2 = ¥260
│        嵐電 幼稚園 ¥130 × 1 = ¥130
│        嵐電 幼児（未満3歳）¥0
│        市バス 大人 ¥230 × 6 = ¥1,380
│        市バス 小学生 ¥120 × 2 = ¥240
│        市バス 幼稚園（未満6歳）¥0 ※同伴幼児は無料
│        9人合計（嵐電+バス）：¥3,510
├─ 購票：嵐電は乗車時・降車時に精算（整理券方式）
│        ICOCA 使用可（嵐電は IC カード対応）
│        市バスは現金（¥230 小銭準備）または ICOCA
├─ 推車友善度：⚠️ 嵐電は旧型車両あり（ステップ 10〜15cm）
│        市バスは低床バスが多い、ベビーカー畳み不要（スペース確保）
│        北野白梅町は嵐電終着駅、段差なし
├─ 電梯位置：嵐電嵐山駅に段差あり（注意）
│            北野白梅町はプラットフォーム低い
├─ 長輩注意：嵐電は座席少ない小型電車（2両、各12m）
│            13:20 発なら混雑はやや落ち着く
└─ 備註：101号 北野白梅町〜金閣寺道 10分間隔運行（秋季増便あり）
         金閣寺道バス停 → 金閣寺 徒歩 約5分
```

## 10 月秋季特別注意事項

- **紅葉状況（10/23）**：嵐山・金閣寺ともに約 20〜30% 色づき。全盛期（11月中旬）ではないが、秋色の始まりを楽しめる。竹林との対比や金閣と部分紅葉の写真は既に価値あり。
- **気温**：15〜22℃。朝夕は薄手のジャケット推薦。幼幼班は防寒レイヤー 1枚追加。
- **週五の人出**：10 月 23 日は平日（金曜）。嵐山の激混みは 11 月以降のため、今の時期は比較的余裕あり。ただし昼 12:00〜14:00 は観光客集中。
- **金閣寺撮影**：14:30〜15:30 が南西からの光で最も金色に輝く時間帯。この時間に合わせたスケジューリングが最適。
- **日没 17:20**：15:30 には金閣寺を出発することで、日没前に移動完了できる。
