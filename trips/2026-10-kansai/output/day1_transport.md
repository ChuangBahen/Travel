## 二、宅急便行李寄送（UKB → 難波飯店）

### 操作流程

```
Step 1：抵達 UKB 入境後
├─ 前往 1F 入境大廳 ヤマト運輸（クロネコヤマト）カウンター
│   位置：UKB 1F 到着ロビー（入境大廳），靠近出口附近
│   ※ 建議出機場前先確認指示牌（黑貓圖示）
│
Step 2：填寫寄送資訊
├─ 寄達地址：難波飯店（OMO7 大阪：大阪市浪速区難波 3-9-15）
│             或 Citadines Namba Osaka：大阪市浪速区難波 1-2-6
├─ 寄送類型：翌日配達（隔日到達，指定 10/21 午前 or 午後）
├─ 個人資料：抵達者姓名 + 飯店電話（建議事先備好日文飯店地址紙條）
│
Step 3：費用支付
├─ 每件費用：¥1,500-2,000（依尺寸 60-100 サイズ）
│   60 サイズ（邊長合計 60cm 以下）：¥1,320 起
│   80 サイズ（邊長合計 80cm 以下）：¥1,540 起
│   100 サイズ（邊長合計 100cm 以下）：¥1,760 起
│   120 サイズ（邊長合計 120cm 以下）：¥1,980 起
├─ 9 人 7 件估算（80-120 サイズ混合）：約 ¥10,500-14,000
└─ 支付方式：現金 / 信用卡均可

Step 4：保留收據
├─ 取得追蹤號碼（追跡番号）
├─ 可用 ヤマト運輸 App 或網站查詢配送狀態
└─ 告知飯店前台行李將於隔日送達（建議 check-in 時說明）
```

### 注意事項

```
重要提醒
├─ 行李僅保留：
│   ├─ 推車（整日使用）
│   ├─ 各人隨身包（零食、雨具、換洗衣物 1 套）
│   ├─ 全素食長輩的隨身零食
│   └─ 幼幼班備用品（尿布、奶嘴等）
├─ 禁止放入宅急便：護照、機票、現金、藥物、充電器（自留）
├─ 若 UKB ヤマト カウンター 未開（罕見），可於三宮駅附近
│   7-Eleven / FamilyMart 使用「コンビニ発送」（便利商店寄送）
└─ 建議事先在台灣準備好：飯店日文地址紙條 × 9 人份
```

## 三、各路段交通詳情

### 段落 1：UKB 神戶機場 → 三宮（Port Liner）

```
Port Liner（ポートライナー）：UKB → 三宮
├─ 移動方式：神戸新交通 Port Liner（自動旅客輸送系統 APM）
├─ 所需時間：約 18 分鐘（全程 8 站）
├─ 費用：
│   成人（12 歲以上）¥340
│   兒童（6-11 歲小學生）¥170
│   幼稚園（4-5 歲，未就學）¥170
│   幼幼班（3 歲以下）¥0（不占座位免費；若需占座 ¥170）
│   ─────────────────────────────────
│   9 人估算：成人 6×¥340 + 兒童 2×¥170 + 幼幼班 1×¥0
│           = ¥2,040 + ¥340 + ¥0 = ¥2,380
│   ※ 若幼幼班計費：+ ¥170 = ¥2,550
├─ 購票：
│   @ UKB 駅 自動售票機（日文 / 英文介面）
│   @ IC 卡（ICOCA / Suica）可直接感應
│   ※ 9 人建議分 2-3 批購票（機器速度快，約 5 分鐘完成）
├─ 推車友善度：⭕
│   ├─ UKB 駅：月台有電梯，推車可直達
│   ├─ 車廂：無固定座位區段，推車可置於車門旁空間
│   ├─ 峰時班距 4-6 分鐘，非尖峰 10-12 分鐘
│   └─ 車廂小（1 節 3 門），9 人帶推車建議等稍空班次
├─ 電梯位置：
│   ├─ UKB 駅：出境大廳 → 地下月台 有電梯（出站右側）
│   └─ 三宮（神戸三宮）駅：月台 → 1F / B1 出口均有電梯
├─ 長輩注意：
│   ├─ 全程坐（有座位），無換車
│   ├─ 路程短（18 分），體力消耗低
│   └─ 三宮駅出口直連各地下街，有遮雨設施
└─ 備註：
    ├─ 運行時間：05:24-23:58（UKB 發），末班充裕
    ├─ Port Liner 延伸線（市民広場方向）注意搭對方向（往三宮）
    └─ 官網：https://www.knt-rc.co.jp/portliner/
```
