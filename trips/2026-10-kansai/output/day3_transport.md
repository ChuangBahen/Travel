## 任務 1｜難波 → 京都 路線比較

### 方案 A｜JR 新快速（推薦首選）

```
JR 大阪環状線 or 御堂筋線（難波 → 大阪）＋ JR 新快速（大阪 → 京都）
├─ 移動方式：地下鐵御堂筋線（難波→梅田）+ JR 新快速（大阪→京都）
│            ※ 若持 ICOCA 可直接刷卡轉乘
├─ 所需時間：難波→梅田 約 8 分鐘 ＋ 大阪→京都 約 28 分鐘 = 合計約 40-50 分鐘
├─ 費用（成人）：御堂筋線 ¥230 + JR ¥580 = 合計 ¥810
│   費用（小學生）：御堂筋線 ¥120 + JR ¥290 = 合計 ¥410
│   費用（幼稚園・幼幼班）：無償（大人同伴，幼兒2名免費）
├─ 購票：ICOCA / Suica 刷卡最便利；現金購票於 JR 大阪駅自動售票機
├─ 推車友善度：⭕ JR 新快速有大空間車廂，乘降月台有電梯
├─ 電梯位置：
│   ・難波（地鐵）→ 梅田：各站均有電梯，梅田1F出口 → JR 大阪駅中央口步行約 5 分鐘
│   ・JR 大阪駅：月台電梯完備
│   ・JR 京都駅：中央口 / 八条口均有電梯
├─ 節慶日注意：時代祭當天（10/22）JR 新快速仍正常運行，但早上大阪→京都方向人潮增加
└─ 備註：JR 新快速每小時 4 班（07:30、08:00、08:30、09:00），節慶日早出發強烈建議
```

### 方案 B｜近鉄京都線

```
近鉄難波線＋近鉄京都線（大阪難波 → 京都）
├─ 移動方式：近鉄（大阪難波 → 丹波橋 → 近鉄京都駅）直通特急
├─ 所需時間：約 35-40 分鐘（特急）/ 約 75 分鐘（急行）
├─ 費用（成人）：¥690（急行・乘車券のみ）/ 特急は別途特急料金 ¥520 加算
│   費用（小學生）：約 ¥350（急行）
│   費用（幼稚園・幼幼班）：無償
├─ 購票：近鉄難波駅自動售票機；特急は指定席要事前購入
├─ 推車友善度：⭕ 近鉄各大站有電梯，特急車廂空間較寬
├─ 電梯位置：大阪難波（近鉄）地下2F ホーム電梯完備；近鉄京都駅亦有電梯
├─ 缺點：
│   ・近鉄京都駅位於烏丸丸太町南方，距御所步行約 15 分鐘
│   ・9 人買特急指定席需提前預訂（否則需分批乘車）
└─ 備註：時代祭當日觀光客多，特急指定席建議前一日網路訂購
```

### 方案 C｜阪急京都本線

```
阪急（梅田 → 河原町）
├─ 移動方式：御堂筋線（難波→梅田）＋ 阪急京都本線（梅田→河原町）
├─ 所需時間：難波→梅田 8 分鐘 + 梅田→河原町 約 43 分鐘（特急）= 合計約 55 分鐘
├─ 費用（成人）：¥230 + ¥410 = 合計 ¥640
│   費用（小學生）：¥120 + ¥210 = 合計 ¥330
│   費用（幼稚園・幼幼班）：無償
├─ 購票：ICOCA / PiTaPa 刷卡；現金購票阪急梅田駅自動售票機
├─ 推車友善度：⚠️ 阪急河原町駅位於地下，有電梯但月台較窄
├─ 電梯位置：阪急梅田（茶屋町口方向）有電梯；河原町駅 2 號出口附近電梯
├─ 優點：終點河原町駅位於三条通旁，時代祭遊行路線正中央，可直接觀賞遊行！
├─ 缺點：距京都御所起點步行約 25 分鐘
└─ 備註：若主要目的是觀賞御池通/三条段遊行，阪急反而最方便
```

### 三方案比較表

| 比較項目 | A JR 新快速 | B 近鉄特急 | C 阪急特急 |
|----------|-------------|------------|------------|
| 難波→京都時間 | 約 50 分 | 約 40 分 | 約 55 分 |
| 9 人總費用（大人6） | ¥4,860 | ¥4,140（急行） | ¥3,840 |
| 終點站距御所距離 | 京都駅→御所 約 3km | 近鉄京都→御所 約 1.5km | 河原町→御所 約 1km |
| 推車友善度 | ⭕ 最佳 | ⭕ 良好 | ⚠️ 尚可 |
| 時代祭觀賞便利性 | 需另乘地鐵或步行 | 近烏丸通（遊行路線） | 直接在三条通旁 |
| 推薦用途 | 最穩健、首選 | 時間優先 | 觀賞御池/三条段遊行 |

**綜合建議：方案 A（JR 新快速）最適合 9 人推車家族**
- 車廂空間大，推車收放最方便
- 京都駅電梯設施完備
- 到達後搭地下鐵烏丸線（2 站）至今出川駅，步行 10 分鐘到御所北門

```

├─ 移動方式：京都市地下鐵烏丸線（北行）
├─ 所需時間：約 10 分鐘（5 站）
├─ 費用：成人 ¥260 / 小學生 ¥130 / 幼兒無償
│   9人合計：¥260×6 + ¥130×1 = ¥1,690
├─ 推車友善度：⭕ 今出川駅有電梯（2 號出口方向）
├─ 電梯位置：今出川駅 1 號出口（御所東側）/ 2 號出口（御所南側）
│   建議出 3 號出口 → 烏丸今出川交差點 → 徒步約 5 分 → 建礼門前
└─ 備註：今出川駅出口即烏丸通，時代祭遊行路線正上方（封路前提早抵達）
```

```
京都御所建礼門前（南門）
```

## 任務 3｜步行路線與推車友善度

### 3-1 京都駅 → 京都御所建礼門

```
路線：京都駅（烏丸口）→ 地下鐵今出川駅 → 御所建礼門前
├─ 移動方式：地下鐵（10 分）＋步行（10 分）= 合計約 20 分鐘
├─ 步行距離：今出川駅 → 建礼門前 約 700m
├─ 路面狀況：⭕ 烏丸通人行道平整，御苑内砂利道但推車可通過

├─ 樹蔭：⭕ 御苑内は大木多く、10 月朝は涼しい
└─ 推薦度：★★★★★（最適合 9 人推車家族）
```

### 3-2 京都駅 → 平安神宮

```
路線：京都駅 → 地下鐵東西線（烏丸御池乗換）→ 東山駅 → 徒步
├─ 移動方式：地下鐵烏丸線（京都→烏丸御池）+ 東西線（烏丸御池→東山）
├─ 地鐵費用：成人 ¥260 / 小學生 ¥130
├─ 東山駅 → 平安神宮：步行約 10 分鐘（仁王門通）
├─ 步行路面：⭕ 仁王門通は舗装路、幅広、推車問題なし
├─ 電梯：東山駅 1 號出口有電梯
├─ 高低差：東山駅（海拔 54m）→ 平安神宮（海拔 52m）= ほぼ平坦
└─ 推薦度：★★★★★（時代祭終點，推車完全無障礙）
```

## 雨天備案（時代祭延至 10/23）

> 若 10/22 降雨，時代祭依官方慣例延至翌日 10/23（週五）

### 衝突問題
10/23（原定嵐山日）與時代祭衝突，需在以下兩個方案中擇一：

| 選項 | 10/22（雨天當日） | 10/23（延期時代祭） |
|------|------------------|---------------------|
| A（推薦） | 改往嵐山（原 10/23 行程提前） | 回京都看延期時代祭 |
| B | 雨天室内觀光（錦市場/京都水族館） | 照常執行嵐山行程，放棄時代祭 |

### 選項 A 詳細（推薦）
- **10/22 雨天改往嵐山**：嵐山天龍寺（竹林有雨天氣氛）、嵐電往返（推車無障礙）
- **10/23 返回京都看時代祭**：從難波再次往京都（交通同上述方案）
- 優點：兩個活動都不放棄
- 缺點：需連續兩天往返難波↔京都，交通費加倍（約 ¥29,000 × 2 天）

### 選項 B
- **10/22 雨天室内**：
  - 錦市場（各種試吃小吃，全天候）
  - 京都鐵道博物館（兒童喜愛，¥1,500/大人）
  - 伏見稻荷（雨天氛圍絕美，但長距離步行）
- **10/23 嵐山照常**：不看時代祭
- 優點：體力負擔較輕
- 缺點：3 年才一次（實際每年舉辦，但家族不常來）的時代祭體驗

### 確認時代祭是否延期
- 官方網站：https://www.heianjingu.or.jp/
- 決定時間：通常當日早上 08:00 前公告
- 建議：前一晚 22:00 前確認隔日天氣，若降雨機率 > 60% 即啟動備案 A
